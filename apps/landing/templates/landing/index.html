Quotes Application

This is the deployment for my final exam in my Python course at Coding Dojo.

I have created a simple application that allows a user to post quotes to a shared database. Each user has a "favorites" list where they can add or remove quotes that have been shared. Users also have a simple profile page that shows the quotes they have contributed.
